#TWVoteData
台灣投票資料

跟隨Ronny的想法，把在[中選會](http://db.cec.gov.tw/histMain.jsp?voteSel=19960301A1)已經有的資料，包裝成JSON格式的形式釋出．
有需求的人一樣可以開issue出來，有空有閒的人可以fork 出來後，再merge進來．

##總統大選格式


	{
		'候選人':[{'總統','副總統'}],
		'投票狀況':{
			'縣市':{
				'鄉鎮市區':{
					'里鎮':{
						'票所',
						'得票數',
						'得票率':[],
						'有效票':[],
						'無效票',
						'投票',
						'已領未投投票',
						'選舉人數',
						'投票率';
					}
				}
			}
		}
	}
